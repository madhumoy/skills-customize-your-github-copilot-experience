# 📘 Assignment: Python Functions

## 🎯 Objective

Learn to define, call, and use functions with parameters and return values in Python. Functions are reusable blocks of code that perform specific tasks, making your programs more organized and efficient.

## 📝 Tasks

### 🛠️ Simple Function

#### Description
Create a simple function that prints a greeting message to the console.

#### Requirements
Completed program should:

- Define a function called `greet`
- The function should print "Hello, World!" when called
- Call the function at least once to demonstrate it works


### 🛠️ Function with Parameters

#### Description
Create a function that accepts a name as a parameter and prints a personalized greeting.

#### Requirements
Completed program should:

- Define a function called `greet_person`
- The function should take one parameter: `name`
- Print "Hello, {name}!" using the provided name
- Call the function with at least two different names to test it


### 🛠️ Function with Return Value

#### Description
Create a function that calculates and returns the square of a given number.

#### Requirements
Completed program should:

- Define a function called `square`
- The function should take one parameter: `num`
- Return the square of the number (num * num)
- Call the function and print the result for at least two different numbers
