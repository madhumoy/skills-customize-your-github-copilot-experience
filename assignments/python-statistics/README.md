# 📘 Assignment: Statistics with Python

## 🎯 Objective

In this assignment, you will learn how to perform basic statistical calculations on a dataset using Python libraries pandas and numpy. You will load data, calculate descriptive statistics, and generate summaries.

## 📝 Tasks

### 🛠️	Load Data and Calculate Basic Statistics

#### Description
Load the provided CSV file containing sample data and calculate the mean, median, and standard deviation for the numerical column.

#### Requirements
Completed program should:

- Import pandas and numpy
- Load the data from 'data.csv' using pandas
- Calculate mean, median, and standard deviation for the 'value' column
- Print the results in a clear format


### 🛠️	Data Summary

#### Description
Generate a statistical summary of the dataset using pandas built-in methods.

#### Requirements
Completed program should:

- Use the describe() method on the DataFrame
- Print the summary statistics
- Include at least mean, std, min, max, and quartiles
