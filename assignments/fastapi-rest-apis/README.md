# 📘 Assignment: Building REST APIs with FastAPI

## 🎯 Objective

Learn to build RESTful APIs using the FastAPI framework in Python. Create a web API with multiple endpoints that handle basic CRUD operations for managing items.

## 📝 Tasks

### 🛠️ Build a REST API with FastAPI

#### Description
Create a FastAPI application that provides REST endpoints for managing a collection of items (like books, tasks, or products). Implement GET, POST, PUT, and DELETE operations with proper request/response handling.

#### Requirements
Completed API should:

- Have a GET endpoint to retrieve all items
- Have a GET endpoint to retrieve a single item by ID
- Have a POST endpoint to create new items
- Have a PUT endpoint to update existing items
- Have a DELETE endpoint to remove items
- Use Pydantic models for request/response validation
- Include proper HTTP status codes and error handling
- Provide interactive API documentation at /docs
