# 📘 Assignment: Python Text Processing

## 🎯 Objective

Students will learn to work with strings, perform file I/O operations, and manipulate text data in Python. This assignment focuses on fundamental text processing skills that are essential for data analysis and application development.

## 📝 Tasks

### 🛠️	String Manipulation

#### Description
Write a Python program that takes a sentence as input from the user and performs various string manipulation operations including reversing the string, counting the number of words, and converting to uppercase.

#### Requirements
Completed program should:

- Accept a sentence input from the user
- Display the reversed version of the sentence
- Count and display the number of words in the sentence
- Convert the entire sentence to uppercase and display it


### 🛠️	File I/O Operations

#### Description
Create a program that reads text from an input file, processes the text by removing punctuation and converting to lowercase, then writes the processed text to an output file.

#### Requirements
Completed program should:

- Read text from a file named 'input.txt'
- Remove all punctuation marks from the text
- Convert all text to lowercase
- Write the processed text to a file named 'output.txt'
